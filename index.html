Naming Rules

Names can contain letters (uppercase and lowercase), numbers, and the symbols _ and $. The first character of the name can’t be a number. JavaScript allows a large variety of characters as variable names, so if you need to use ñ, ö or even π, you can, but is highly discouraged because others might not be able to use these characters and then your website will look broken.

NOTA importante

Notice JavaScript is case sensitive, so capital letters do make a difference. For example, color and Color are different variable names. Although we said you could use any English or non-English word, still there’s a bit constraint: there are some reserved keywords that can’t be used, for example: let, class, return, and function are reserved

To use the .slice() method you need to add in the () of the method, the two indexes that will mark the start and end position of the new string you wish to extract, separated by a comma. Note that the index indicated in the end position will not be included.
